# University-Enrollment-System
A Student Friendly University Enrollment System made for the academic course COMP 6905: Software Engineering at MUN. Made using Python, JavaScript, Django, HTML and CSS; the features include chatbot support, language localization features and user feedback system along with automated testing features.
